# Bulk Download for Adobe Podcast — Enhance Speech

Landing page for the **Bulk Download for Adobe Podcast** Chrome extension — the
one-click way to **download all your Adobe Enhance Speech files at once**, with
original filenames, instead of clicking each file one by one.

🔗 **Live site:** <https://pfehringer4.github.io/adobe-podcast-bulk-download-site/>

## What it solves

Adobe Podcast (Enhance Speech) lets you bulk **upload** audio, but only lets you
**download** the enhanced files one at a time. Run 40, 100, or 500 files through
Enhance Speech and you face hundreds of clicks. This Chrome extension adds the
missing **“Download all”** button: one click downloads every completed enhanced
file with its original name, with a progress bar and auto-retry.

- ✅ One-click bulk / batch download for Adobe Enhance Speech
- ✅ Keeps original filenames
- ✅ Progress bar + auto-retry
- ✅ Runs locally — your own signed-in session, your own files, nothing sent to a server
- ✅ Free up to 5 files per run · Pro unlimited, one-time €39

It automates the clicking on your own files in your own session — it does **not**
bypass any Adobe download limits.
